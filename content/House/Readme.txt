Here is a array house example with start and end cap.
You can add more floors by increase the count in the array modifiers.

Animated No
Rigged No
Lowpoly (game-ready) Yes
Geometry Polygon mesh
Polygons 2,438
Vertices 3,047
Textures Yes
Materials Yes
UV Mapping Yes
Unwrapped UVs Non-overlapping

Have fun with it and feel free to use this house example for your project :)
